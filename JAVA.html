Java Program Example

public class HelloWorld {
    public static void main(String[] args) {
        // Print Hello World to the console
        System.out.println("Hello, World!");
    }
}

Explanation:

The above program is a simple Java program that prints "Hello, World!" to the console.

  • public class HelloWorld: Declares a class named HelloWorld.
  • public static void main(String[] args): Main method where execution begins.
  • System.out.println: Prints the message to the console.
